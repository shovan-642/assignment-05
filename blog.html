Important FAQ

What is the Document object model (DOM)?

The Document Object Model (DOM) is a way to represent a web page. It allows programs to change the content and structure of the page. The DOM makes web pages interactive.

how do you select an element from the DOM?

we can use: document.getElementbyId or document.getElementsbyTagName

What is event delegation in the context of the DOM, and why is it useful?

Event delegation means adding one event listener to a parent. It handles events for all children. This saves time and makes it easier.

How do you manipulate an element's attributes and styles using the DOM?

Update an element's properties like src for an image. Modify an element's CSS, like setting the color to blue.
